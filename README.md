# Bleepay Technical Documentation

Technical documentation for the **Bleepay** project built with [Mintlify](https://mintlify.com).

## Structure

- **index.mdx** — homepage
- **getting-started/** — introduction, quickstart, and core concepts
- **integration/** — overview, SDK, workflow, and webhooks
- **e-commerce/** — overview, checkout, and benefits
- **web3/** — developers, wallet providers, dApps, and payment processing
- **architecture/** — execution flow, off-ramp, security, dependencies, and fees
- **reference/** — terminology, protocol flow, and voucher modes
- **images/** — diagrams (PNG)
- **docs.json** — Mintlify configuration (navigation, colors, and theme)

## Local Development

Requirements: Node.js v20.17.0+

```bash
# Install the Mintlify CLI
npm install -g mint

# Start the local preview server
mint dev
```

Dev env is available at: **http://localhost:3000**
