title: "Delete Webhook Endpoint"
icon: "angle-right"
---

> Permanently delete a webhook endpoint.

`POST /api/v1/businesses/:businessId/webhook-endpoints/:endpointId/delete-endpoint`

Deleted endpoints cannot be recovered. Any pending deliveries for this endpoint are cancelled.

## Parameters

### Path

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `businessId` | `string` | Yes | The business ID |
| `endpointId` | `string` | Yes | The endpoint ID |

## Request example

```shell
curl --request POST 'https://payments.bleepay.com/api/v1/businesses/biz_abc/webhook-endpoints/wh_abc123/delete-endpoint' \
  --header 'Authorization: Bearer <business_token>' \
  --header 'Content-Type: application/json' \
  --data '{}'
```

## Response

Returns HTTP 200 with an empty body.
