> Manually retry a failed webhook delivery.

`POST /api/v1/businesses/:businessId/webhook-deliveries/:deliveryId/retry-delivery`

Triggers an immediate retry of a previously failed delivery. This is in addition to the automatic retry schedule (up to 3 retries at 30-second intervals).

## Parameters

### Path

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `businessId` | `string` | Yes | The business ID |
| `deliveryId` | `string` | Yes | The delivery ID |

## Request example

```shell
curl --request POST 'https://payments.bleepay.com/api/v1/businesses/biz_abc/webhook-deliveries/dly_abc123/retry-delivery' \
  --header 'Authorization: Bearer <business_token>' \
  --header 'Content-Type: application/json' \
  --data '{}'
```

## Response

Returns HTTP 200 with an empty body.
